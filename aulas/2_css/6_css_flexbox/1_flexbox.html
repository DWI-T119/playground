FlexBox

Dividir em duas categorias:

  • Container (elemento pai)
  • Items (elementos filhos)

  • display: flex

    Indica que o container utilizará o posicionamento flexbox

  • flex-direction

    Indica a direção que os items serão disponibilizados

  • flex-wrap

    Indica se os items quebrarão de linha ao diminuir o container

  • flex-flow

    Atalho para flex-direction + flex-wrap

  • justify-content

    Usado para alinhar os items

  • align-items

    Usado para alinhar os items

  • align-content

    Usado para alinhar as linhas
