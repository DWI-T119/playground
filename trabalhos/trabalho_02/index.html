#Instruções

  • Criar uma pasta, chamada trabalho_02, dentro da pasta que contenha seu nome aqui no projeto geral (Exemplo: alunos/meu_nome/trabalho_02)
  • Após criar a pasta, comece a enviar as alterações para o GitHub
  • O trabalho inteiro será apenas um site, porém com várias páginas
  • Cada objetivo terá sua pasta e seu arquivo
  • Organizem os objetivos como quiserem dentro do "site" (página inicial e menu serão livres), porém criem de uma forma acessível. Exemplo, uma página principal com os links:
    • Objetivo 01 - CSS Básico
    • Objetivo 02 - CSS Flex
    • ...
    • Objetivo 05 - Javascript Básico

#Data, como entregar e dúvidas

  • Data:
    • Não se preocupem com a data de entrega, vamos terminar e aprender juntos como um exercício. Porém quanto mais eficientes formos, mais longe poderemos ir no conhecimento dentro da disciplina e aí começaremos projetos mais reais depois dessa introdução de HTML, CSS e JS.
  • Como Entregar:
    • Enviar o link da sua página pelo Moodle
  • Dúvidas e Ajudas:
    • Qualquer dúvida, tanto sobre o conteúdo como sobre o trabalho, só pedir ajuda (email, wtsapp etc) que responderei assim que possível. Isso inclui, principalmente, ajuda para resolver os objetivos.

#CSS (2 pontos)

Serão avaliados os seguintes pontos, contabilizando também a pagina inicial e menu geral que serão livres:

  • CSS Básico (1)

    • Propriedades de Texto (font-size, color etc)
  • Flexbox (0,5)
    • Adaptar e organizar items dentro de um Container Flex
  • CSS Grid (0,5)
    • Adaptar e organizar items dentro de um Container Grid
